Internal Business Automation & API Management Tool
MKey Manager is a proprietary middleware and configuration handler designed exclusively for Hoopstreet Sports Apparel Accessories Shop (DTI No. 7918311). This repository serves as the central hub for managing API integrations, OAuth2 redirect flows, and internal business logic for TikTok Shop Philippines.
Overview
This tool is built to streamline the operations of Hoopstreet by automating the following:
• Product Synchronization: Fetching live product data and price updates from our authorized TikTok Shop.
• Content Automation: Orchestrating video upload drafts and metadata for affiliate marketing campaigns.
• OAuth2 Management: Handling secure access_token exchanges for our internal n8n automation workflows.
